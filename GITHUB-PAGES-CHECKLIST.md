# GitHub Pages Checklist

## Publish

1. Upload the contents of `legal-site-github-pages-package/` to the repository root.
2. Commit to `main`.
3. Enable GitHub Pages from the `main` branch root.

## Verify

1. Open the HTTPS home page.
2. Open `/404.html`.
3. Open every app privacy page.
4. Open every app support page.
5. Open every app terms page.
6. Confirm `.nojekyll` exists in the published root.

## After publish

1. Fill [legal-site-published-url-matrix-template.md](/Users/pengpeng/Desktop/codex工作区/AppProjects/legal-site-published-url-matrix-template.md).
2. Paste those exact URLs into App Store Connect.
3. If needed, paste the final terms URLs into each app's `review-notes.md`.
