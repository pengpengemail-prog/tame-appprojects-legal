# tame-appprojects-legal

Static GitHub Pages repository for the TAME AppProjects legal site.

## Publish

1. Create a public GitHub repository named `tame-appprojects-legal`.
2. Copy every file in this directory to the repository root.
3. Commit to `main`.
4. Enable GitHub Pages from the `main` branch root.

## Expected URL pattern

- `https://<your-pages-domain>/sleeptracker/privacy.html`
- `https://<your-pages-domain>/habitforge/privacy.html`
- `https://<your-pages-domain>/recipevault/privacy.html`
- `https://<your-pages-domain>/flashmaster/privacy.html`
- `https://<your-pages-domain>/moneymind/privacy.html`
