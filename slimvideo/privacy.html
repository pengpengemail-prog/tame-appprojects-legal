TAME SlimVideo Privacy Policy

TAME SlimVideo is a local media organization utility.

The app helps review local media items on device. It does not sell personal data, does not use ad tracking, and only changes media after the user confirms an action.

Data handling

  • Version 1.0 does not require an account for the main app experience.
  • User-created content and settings are stored locally on the device unless a system feature says otherwise.
  • The app does not use third-party advertising SDKs or cross-app tracking.
  • Diagnostics, if provided by Apple system services, are used only to improve stability.
