TAME SlimVideo Support

TAME SlimVideo helps you compress large videos locally and save smaller copies back to Photos.

Common Checks

  • Start a new scan from Dashboard if newly recorded videos do not appear.
  • Save the compressed copy before removing an original video.
  • Keep the original if you want to compare quality before deleting anything.
